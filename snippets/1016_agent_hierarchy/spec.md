# Sudoku Multi-Agent Demo

## Goal

Demonstrate hierarchical AI agent collaboration.

## Architecture

1 controller agent
3 worker agents

## Technical Stack

- TypeScript
- Node.js
- Console application

## Features

- Sudoku board
- Sudoku validator
- Simple backtracking solver
- Console rendering
- Basic tests

## Non-Goals

- Fancy UI
- Advanced puzzle generation
- Performance optimization
